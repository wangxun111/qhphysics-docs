# Force Report Create
| **INC-07** | **DocBuilder 脚本截断**<br>HTML未更新 | **原因**: LLM 生成长脚本时在 52 行处意外中断，导致转换逻辑缺失。<br>**影响**: 构建器静默失败，网页不更新。 | **修复**: 重新生成完整脚本并人工校验文件末尾。 | **规范**: 生成超过 50 行的脚本后，必须使用 `Get-Content -Tail 5` 校验完整性。 |
| **INC-08** | **Mermaid 连线消失**<br>流程图断点 | **原因**: 隐式定义节点 (`A --> B`) 后再定义形状 (`B{Label}`) 导致渲染冲突。<br>**影响**: 连线在网页端不可见。 | **修复**: 强制在第一次引用节点时显式定义形状 (`A --> B{Label}`)。 | **规范**: 禁止隐式定义。所有节点形状必须在首次出现时完整声明。 |
