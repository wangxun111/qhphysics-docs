## 📝 笔记功能

两个网页都配备相同的笔记系统：

1. **添加笔记**
   - 将鼠标悬停在任何内容块上
   - 点击右上角的 ✏️ 按钮
   - 输入笔记内容
   - 点击"保存"

2. **管理笔记**
   - 右侧"📝 笔记"面板显示所有笔记
   - 点击笔记项自动跳转
   - 可以编辑、删除笔记
   - 笔记自动保存到本地浏览器 (localStorage)

3. **笔记隔离**
   - XunPhysics 的笔记存储在 `qhphysics_notes`
   - PhysX 的笔记存储在 `physx_analysis_notes`
   - 两个文档的笔记不会互相干扰
